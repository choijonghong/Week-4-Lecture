### 텐서플로우 플레이그라운드

* 텐서플로우 플레이그라운드는 신경망의 학습 과정을 시각적으로 보여주는 웹 기반 도구입니다.

* 사용자는 학습률, 활성화 함수 등 다양한 설정을 변경하여 모델에 미치는 영향을 실시간으로 확인합니다.

* 이는 머신러닝의 기본 개념을 쉽게 이해하고 실험해 볼 수 있도록 돕는 교육용 시뮬레이터입니다.
